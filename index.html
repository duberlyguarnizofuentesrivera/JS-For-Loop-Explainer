Aprendiendo a usar el bucle FOR

Con un pequeño script de explicación

Instrucciones:

Ingresa los datos solicitados en los cuadros, y luego el bucle se completará con los mismos.

Iterador     
Valor inicial
Valor final  
Incremento

Selecciona el tipo de relacion entre el iterador y el valor final:

for (int i = 0; i < 5; i++){
    // Muestra valor del iterador
    System.out.println(i);
}
