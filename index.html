Aprendiendo a usar el bucle FOR

Con un pequeño script de explicación

Instrucciones:

Ingresa los datos solicitados en los cuadros, y luego el bucle se completará con los mismos. El primer cuadro es para el nombre de la variable que hará de iterador. El segundo cuadro, para el numero (entero) de inicio del bucle. El tercer cuadro, para el numero final. Nota que el valor de inicio puede ser mayor, igual o menor que el valor final. El ultimo cuadro es para marcar el paso del bucle (cuanto avanzará o retrocederá el iterador en cada bucle). Finalmente, selecciona a la derecha el sentido del operador de comparación. Las combinaciones ilegales, así como los bucles infinitos (buffer overflow, o desbordamiento de pila) serán notificados. ¡Juega con los valores!
